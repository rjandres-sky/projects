Instruction

  • Each player has 49 game pieces equally distributed to each holes - except the head which remain empty.
  • For the first turn, a player may decide which of the seven holes wishes to begin with.
  • The last piece will dictate whether to continue the turn.
  • If landed to home/head, pick another holes to begin.
  • If landed to not empty hole, all piece will pick and continue to distribute.
  • If landed to an empty hole, current player turns end.
  • The game ends when 1 player has a score above 49.
